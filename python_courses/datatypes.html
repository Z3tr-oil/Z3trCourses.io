Python Data Types

Learn about different data types in Python.

Overview

Python has several built-in data types that are used to store values. Understanding these types is fundamental to programming in Python.

Integers (int)

Whole numbers, positive or negative, without decimals.


x = 10
y = -5
print(type(x))  # <class 'int'>
  • Can be arbitrarily large in Python 3.
  • Common operations: +, -, *, //, %, **

Floating Point Numbers (float)

Numbers with a decimal point.


pi = 3.14159
temperature = -7.5
print(type(pi))  # <class 'float'>
  • Can represent very large or very small numbers.
  • Be aware of floating-point precision issues.

Strings (str)

Sequences of Unicode characters, used to store text.


name = "Alice"
greeting = 'Hello, world!'
multiline = """This is
a multiline string."""
print(type(name))  # <class 'str'>
  • Strings are immutable (cannot be changed after creation).
  • Common methods: .lower(), .upper(), .replace(), .find()

Booleans (bool)

Logical values: True or False.


is_active = True
is_admin = False
print(type(is_active))  # <class 'bool'>
  • Often used in conditions and comparisons.
  • Result of logical expressions: 3 > 2 is True

Lists (list)

Ordered, mutable collections of items (can be of mixed types).


numbers = [1, 2, 3, 4]
mixed = [1, "two", 3.0, True]
print(numbers[2])  # 3
numbers.append(5)
print(numbers)  # [1, 2, 3, 4, 5]
  • Lists can be nested: matrix = [[1,2],[3,4]]
  • Common methods: .append(), .remove(), .pop(), .sort()

Tuples (tuple)

Ordered, immutable collections of items.


coordinates = (10, 20)
colors = ("red", "green", "blue")
print(coordinates[0])  # 10
  • Tuples are immutable (cannot be changed).
  • Useful for fixed collections of items.

Dictionaries (dict)

Unordered collections of key-value pairs.


person = {"name": "Alice", "age": 30}
print(person["name"])  # Alice
person["city"] = "London"
print(person)  # {'name': 'Alice', 'age': 30, 'city': 'London'}
  • Keys must be unique and immutable (e.g., strings, numbers, tuples).
  • Common methods: .keys(), .values(), .items(), .get()

Sets (set)

Unordered collections of unique items.


unique_numbers = {1, 2, 3, 2, 1}
print(unique_numbers)  # {1, 2, 3}
unique_numbers.add(4)
print(unique_numbers)  # {1, 2, 3, 4}
  • Useful for removing duplicates.
  • Common methods: .add(), .remove(), .union(), .intersection()

Type Conversion


x = 5
y = str(x)      # "5"
z = float(x)    # 5.0
n = int("42")   # 42
