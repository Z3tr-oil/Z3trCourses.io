Python Arrays (Lists)

Working with lists in Python

What is a List?

Lists are Python's built-in data structure for storing ordered collections of items. Lists can contain elements of different types. 


numbers = [1, 2, 3, 4, 5]
fruits = ["apple", "banana", "cherry"]
mixed = [1, "two", 3.0, True]

Accessing and Modifying Lists


print(fruits[0])      # apple
fruits[1] = "orange"
print(fruits)         # ['apple', 'orange', 'cherry']

List Methods


numbers.append(6)
print(numbers)        # [1, 2, 3, 4, 5, 6]
numbers.remove(3)
print(numbers)        # [1, 2, 4, 5, 6]
print(len(numbers))   # 5
numbers.insert(2, 99)
print(numbers)        # [1, 2, 99, 4, 5, 6]
numbers.sort()
print(numbers)        # [1, 2, 4, 5, 6, 99]

List Slicing


nums = [10, 20, 30, 40, 50]
print(nums[1:4])   # [20, 30, 40]
print(nums[:3])    # [10, 20, 30]
print(nums[::2])   # [10, 30, 50]

Looping Through a List


for fruit in fruits:
    print(fruit)

List Comprehensions


squares = [x*x for x in range(5)]
print(squares)  # [0, 1, 4, 9, 16]

Nested Lists


matrix = [
    [1, 2, 3],
    [4, 5, 6]
]
print(matrix[1][2])  # 6
Tip: Lists are mutable. You can change, add, or remove elements after creation.
