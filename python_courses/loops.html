Python Loops

Repeating actions with for and while loops

for Loops

Use for to iterate over a sequence (list, tuple, string, etc.):


for i in range(5):
    print(i)
# Output: 0 1 2 3 4

Looping Through a List


fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Looping Through a String


for char in "hello":
    print(char)

while Loops

Repeat as long as a condition is true:


count = 0
while count < 3:
    print("Counting:", count)
    count += 1

Breaking and Continuing


for i in range(10):
    if i == 5:
        break  # Exit the loop
    if i % 2 == 0:
        continue  # Skip even numbers
    print(i)

Nested Loops


for i in range(1, 4):
    for j in range(1, 4):
        print(i, j)

Using else with Loops


for i in range(3):
    print(i)
else:
    print("Loop finished!")
Tip: Use break to exit a loop early, continue to skip to the next iteration.
