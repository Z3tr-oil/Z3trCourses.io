How to display output in Python

What is print()?

The print() function is used to display output to the screen. It can print strings, numbers, variables, and even the results of expressions.

Basic Usage


print("Hello, World!")
print(42)
print(3.14)
Tip: You can print multiple items by separating them with commas.

name = "Alice"
age = 30
print("Name:", name, "Age:", age)

Printing Expressions


print(2 + 3)         # 5
print("Sum:", 2 + 3) # Sum: 5

Formatting Output

  • f-strings (Python 3.6+):

name = "Bob"
score = 95
print(f"{name} scored {score} points.")
  • Using .format():

print("{} scored {} points.".format(name, score))

Printing Without Newline

By default, print() adds a newline at the end. Use end="" to change this:


for i in range(3):
    print(i, end=" ")
# Output: 0 1 2

Printing Special Characters


print("Line1\nLine2")   # \n is a newline
print("Tab\tSpace")     # \t is a tab

Escape Characters


print("He said \"Hello\"")  # Prints: He said "Hello"
