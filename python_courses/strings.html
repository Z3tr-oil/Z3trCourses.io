Python Strings

Working with text in Python

Creating Strings


s1 = "Hello"
s2 = 'World'
s3 = """This is a
multiline string"""

String Operations

  • Concatenation: +
  • Repetition: *
  • Length: len()
  • Indexing: []
  • Slicing: [start:end]
  • Membership: in

greet = "Hello"
name = "Alice"
message = greet + ", " + name  # Concatenation
print(message)  # Hello, Alice
print(greet * 3)  # HelloHelloHello
print(len(name))  # 5
print(name[0])    # A
print(name[1:4])  # lic
print("A" in name)  # True

String Methods


text = "  python is Fun!  "
print(text.strip())      # Remove whitespace
print(text.lower())      # python is fun!
print(text.upper())      # PYTHON IS FUN!
print(text.replace("Fun", "Awesome"))  # python is Awesome!
print(text.find("is"))   # 9
print(text.startswith("  py")) # True
print(text.endswith("!  "))    # True

Immutability

Strings cannot be changed after creation. All string methods return new strings.


s = "hello"
s2 = s.upper()
print(s)   # hello
print(s2)  # HELLO

String Formatting


name = "Bob"
age = 25
print(f"My name is {name} and I am {age} years old.")
print("My name is {} and I am {} years old.".format(name, age))
Tip: Use split() and join() to work with words and lists.

sentence = "one two three"
words = sentence.split()
print(words)  # ['one', 'two', 'three']
joined = "-".join(words)
print(joined) # one-two-three
