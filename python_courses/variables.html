Python Variables

Understanding variables in Python.

What are Variables?

  • Variables are containers for storing data values.
  • Python has no command for declaring a variable.
  • A variable is created the moment you first assign a value to it.
  • Variable names are case-sensitive and must start with a letter or underscore.

Examples


x = 5
y = "Hello"
z = 3.14
print(x)      # 5
print(y)      # Hello
print(z)      # 3.14

Variable Naming Rules

  • Must start with a letter or underscore (_)
  • Cannot start with a number
  • Can only contain alphanumeric characters and underscores (A-z, 0-9, _)
  • Case-sensitive: age and Age are different
  • Cannot use reserved words (e.g., class, for, if)

Valid and Invalid Variable Names


# Valid
my_var = 10
_var2 = "ok"
userName = "Alice"

# Invalid (will cause errors)
# 2var = 5
# my-var = 7
# class = "reserved word"

Multiple Assignment


a, b, c = 1, 2, 3
print(a, b, c)  # 1 2 3

x = y = z = 0
print(x, y, z)  # 0 0 0

Changing Variable Values


score = 10
print(score)  # 10
score = 20
print(score)  # 20

Dynamic Typing

Python variables can change type after assignment:


var = 5
print(type(var))  # <class 'int'>
var = "Now I'm a string"
print(type(var))  # <class 'str'>

Swapping Variables


a = 1
b = 2
a, b = b, a
print(a, b)  # 2 1

Unpacking Lists/Tuples


data = [10, 20, 30]
x, y, z = data
print(x, y, z)  # 10 20 30

point = (5, 7)
x, y = point
print(x, y)  # 5 7

Deleting Variables


x = 10
del x
# print(x)  # This will cause an error: NameError: name 'x' is not defined

Global and Local Variables


x = 5  # global variable

def foo():
    y = 10  # local variable
    print(x, y)

foo()  # 5 10
print(x)  # 5
# print(y)  # Error: y is not defined outside foo()
