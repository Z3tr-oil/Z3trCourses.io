Python Syntax

Learn about the basic syntax of Python programming language.

Python Syntax Overview

  • Python uses indentation (whitespace) to define code blocks, not braces {}.
  • Statements end with a newline, not a semicolon ; (though semicolons are allowed).
  • Case sensitive: Variable and variable are different.
  • Comments start with #.
  • Strings can be enclosed in single (') or double (") quotes.

Example


# This is a comment
print("Hello, World!")
