Getting user input in Python

The input() Function

The input() function allows you to get input from the user as a string.


name = input("Enter your name: ")
print("Hello,", name)
Tip: The prompt inside input() is optional.

x = input()

Converting Input Types

By default, input() returns a string. Use int() or float() to convert: 


age = input("Enter your age: ")
age = int(age)
print("Next year you will be", age + 1)

Input and Expressions


a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
print("Sum is", a + b)
print("Product is", a * b)

Reading Multiple Values


# Input: 10 20
x, y = input("Enter two numbers: ").split()
x = int(x)
y = int(y)
print(x, y)

Input in Loops


for i in range(3):
    name = input("Enter a name: ")
    print("Hello,", name)
