Python If Statements

Making decisions in Python

Basic if Statement


x = 10
if x > 5:
    print("x is greater than 5")

if-else Statement


age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

if-elif-else Statement


score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: D or F")

Nested if Statements


num = 7
if num > 0:
    if num % 2 == 0:
        print("Positive even number")
    else:
        print("Positive odd number")
else:
    print("Zero or negative")

Short-Hand If and If-Else


a = 5
b = 10
if a < b: print("a is less than b")

# Short-hand if-else (ternary)
result = "even" if a % 2 == 0 else "odd"
print(result)

Boolean Expressions in If


x = 7
if 0 < x < 10:
    print("x is a single-digit positive number")
Tip: Indentation is required for code blocks in Python!
