Mathematical & Logical Operations

Learn how to use math and logic in Python

Mathematical Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division (float result)
  • // Floor Division (integer result)
  • % Modulus (remainder)
  • ** Exponentiation (power)

a = 7
b = 3
print(a + b)   # 10
print(a - b)   # 4
print(a * b)   # 21
print(a / b)   # 2.333...
print(a // b)  # 2
print(a % b)   # 1
print(a ** b)  # 343
Tip: Use abs() for absolute value, round() for rounding, and pow() for powers.

print(abs(-5))      # 5
print(round(3.14159, 2))  # 3.14
print(pow(2, 4))    # 16

Logical Operators

  • and - True if both are True
  • or - True if at least one is True
  • not - Inverts the value

x = True
y = False
print(x and y)  # False
print(x or y)   # True
print(not x)    # False

Comparison Operators

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

a = 5
b = 10
print(a == b)  # False
print(a != b)  # True
print(a > b)   # False
print(a < b)   # True

Combining Math and Logic


x = 7
if x % 2 == 0 and x > 0:
    print("x is a positive even number")
else:
    print("x is odd or negative")

Using the math Module


import math
print(math.sqrt(16))   # 4.0
print(math.sin(math.pi/2))  # 1.0
